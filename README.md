# Holmes Lab Documentation

[https://holmeslab.github.io/holmeslab/](https://holmeslab.github.io/holmeslab/)

This is a site to hold the documentation for the Holmes Lab at Rutgers University. There are over 100 pages of tutorials, resources and statement-of-procedures for the Holmes Lab. Created by Kaley Joss, 2025.

This wiki can be added to by anyone in the Holmes Lab Github organization. For instructions on how to update this Wiki, visit: [Updating This Wiki](https://holmeslab.github.io/holmeslab/docs/wiki/add-to-wiki/).
